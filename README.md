# Carte_web_ugb
Ceci est une carte interactive de l’Université Gaston Berger (UGB), réalisée et mise en ligne par la promotion 32 de Géographie pour l’année académique 2025/2026 sous l’encadrement du Dr Dah Dieng. Elle présente une visualisation géographique du campus et met en valeur ses différents espaces universitaires.
